Martikulasi Academic Room

Program academic room dirancang untuk mahasiswa Sistem Informasi Kota Cerdas tentang pengenalan tentang dunia dasar dasar pembuatan website dengan menggunakan HTML (Hyper Text Markup Language) untuk menyusun halaman web. Dalam sesi martikulasi mahasiswa sistem informasi kota cerda di ajarkan memahami cara membuat struktur halaman sederhana menggunakan HTML, mulai dari membuat Heading, paragraf hingga menambahkan gambar di halaman

Selain teori mahasiswa juga di ajak mempraktikan pembuatan halaman web sederhana, setiap mahasiswa yang mengikuti academic room belajar menulis kode HTML dari awal hingga menghasilkan halaman yang diakses melalui browser. materi juga mencakup pengenalan tentang tag html

    Daftar materi paling favorit

    1. sejarah HTML
    2. instal extension
    3. pengenalan tag HTML
    4. instal github
    5. membuat halaman web
